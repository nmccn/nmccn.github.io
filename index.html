Sensor System Bop-It Clone

Working with a small team, designed and implemented a game in which various LEDs indicate a sensor the user must activate to continue. This task was accomplished using an RPi and Python.

  • Demo Video

Learning Universal Remote

Developed routines to receive, clone, and re-transmit the IR signals of a donor remote. Realized using a PIC microcontroller and C programming, on an entirely hand-assembled circuit board.

  • Source Code
  • Coming Soon

Spanish Workbook Mobile App

Alongside a local school, produced a mobile application based on their physical workbook. Responsible for the splashscreen, navigation, and activity templates, utilizing Dart & the Flutter SDK.
